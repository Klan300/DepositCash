# Require
- [x] JDK 1.8 or later
- [x] Apache Maven
- [x] IDE (suggest intelliJ)

# Installation

1. git clone
```bash
git clone https://github.com/Klan300/DepositCash.git
```

2. cd to DepositCash
```bash
cd DepositCash
```

3. Open in `IDE`

4. Install Dependencies in `pom.xml`

# Running Test

- install Maven
- go to `src/test`
```bash
cd src/test
```
- Run test
```bash
mvn test
```
